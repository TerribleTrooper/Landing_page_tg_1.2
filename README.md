Запустить сервисы:
- docker compose up

Установить зависимости:
- pip install -r requirements.txt

Хранение данных: Все заявки сохраняются в базе данных postgesSQL

Используемые технологии:
- FastAPI
- Pydantic
- Uvicorn
- Docker
- Nginx
- HTML / CSS / JS

Планы развития: Возможные улучшения:
- Админ-панель
- Авторизация
- Telegram-уведомления
- Аналитика
- CRM-интеграция
